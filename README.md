# Учебный проект: Внедрение CI/CD (GitHub Actions) и Blue-Green Deploy на [Vercel](https://vercel.com)

**Цель проекта:** Практическая настройка полного цикла непрерывной интеграции и доставки (CI/CD) с использованием GitHub Actions и стратегии Blue-Green Deploy на платформе Vercel.

**Краткое описание:** В качестве базового приложения использована система учета оборудования компании. Основная фокусная задача проекта — не функциональность приложения, а корректная настройка и демонстрация рабочего пайплайна развертывания.

## Стек технологий и инструментов
*   **Приложение:** ASP.NET Core
*   **CI/CD:** GitHub Actions
*   **Хостинг и деплой:** Vercel (Blue-Green стратегия)
*   **База данных:** PostgreSQL от [Neon](https://neon.com/) (одного из поставщиков баз данных Vercel)

## Что было успешно реализовано
1.  Настроен автоматический пайплайн сборки и тестирования в **GitHub Actions**.
2.  Реализована стратегия **Blue-Green Deploy** для развертывания на **Vercel**, обеспечивающая минимальное время простоя при обновлениях.
3.  Пайплайн полностью рабочий и готов к использованию.

## Примечание о состоянии кода приложения
Само веб-приложение (учет оборудования) является учебной основой и **имеет незавершенную функциональность и известные баги** (в основном, связанные с миграциями БД и модулями тестирования). Основная ценность репозитория — конфигурация CI/CD.
